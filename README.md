## 简介
将RAG模块从chatchat项目单独抽出来立项，各项功能模块有独立接口，支持一对一、一对多的文本相似度计算，文档向量化存储，输入问题召回文档，重排，以及添加所有可检索增强的技术方案。
作为独立模块，也是因为后续接大模型的时候，更有灵活性。
